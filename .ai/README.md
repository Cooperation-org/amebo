# .ai/ — working aids for coding agents

Entry point for this repo is [../CLAUDE.md](../CLAUDE.md) (pointed to by [../AGENTS.md](../AGENTS.md)). This directory holds the aids agents use **while working**, as opposed to architecture docs (those live in [../docs/](../docs/), human-readable) and amebo's own runtime skills (those are product content, not agent guidance — see CLAUDE.md).

| File | Use it when |
|---|---|
| [review-checklist.md](review-checklist.md) | Before every commit, and when reviewing anyone's diff |

Add new aids here (and list them above) rather than scattering instructions elsewhere. Keep each aid DRY: point to the governing doc, don't restate it.
