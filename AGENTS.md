# Agent entry point

**All agent guidance for this repo starts at [CLAUDE.md](CLAUDE.md).** (Named for the tool that first used it; it is vendor-neutral — whatever agent you are, read it.)

From there you'll be pointed to:
- [`.ai/`](.ai/README.md) — working aids for agents (review checklist, conventions)
- [`docs/`](docs/) — architecture docs (human- and agent-readable)
- [`scratch.md`](scratch.md) — the live cross-session coordination board (read before working; append, never delete)

Do not create agent instructions anywhere else — extend these.
